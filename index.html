Use keys QWER to manipute the orientation of a selected control point.

For loading point files, each point should consist of 6 numbers, splitted by space, that represent the position and the orientation. A minimum of four points are required.
e.g.
120 10 120 0 1 0
120 20 -120 0 1 0
-120 30 120 0 1 0
-120 40 -120 0 1 0
120 50 120 0 1 0
120 60 -120 0 1 0
-120 70 120 0 1 0
-120 80 -120 0 1 0
